# carsads

Car classifieds — built on [@proappstore/sdk](https://github.com/proappstore-online/platform).

**Live:** https://carsads.proappstore.online

## Features

- GitHub sign-in (shared identity with FreeAppStore)
- Browse car listings with filters (make, model, year, price, location)
- Listing detail with photo gallery, specs, and embedded map
- Post a listing (Pro subscribers) — upload photos, fill specs
- Manage your listings (edit, mark sold, delete)
- Save favorites to revisit later

## Dev

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## Deploy

Push to main — auto-deploys via Cloudflare Pages.
